📊 Visión General del Sistema

TRAKWERK PARQUES es una aplicación web moderna SPA construida con tecnología serverless y cloud-native.

  • Catálogo de 21 lotes industriales en 5 parques
  • Búsqueda y filtrado avanzado
  • Cotizaciones en tiempo real vía WhatsApp
  • Escalabilidad automática
  • Disponibilidad 99.9% en producción

🔩 Capas de Arquitectura

1️⃣ CAPA DE PRESENTACIÓN (Frontend)

Tecnología: React 19 + Next.js 16 + TypeScript + Tailwind CSS

Componentes: Login, Dashboard, Catálogo, Parques, Cotizador

2️⃣ CAPA DE APLICACIÓN (Framework)

Tecnología: Next.js 16 App Router

Características: Server Components, Middleware de autenticación, TypeScript strict

3️⃣ CAPA DE API & INTEGRACIÓN

Tecnología: Supabase API + WhatsApp Integration

Servicios: CRUD de datos, Autenticación JWT, Cotizaciones

4️⃣ CAPA DE BASE DE DATOS

Tecnología: PostgreSQL (Supabase)

Tablas: parques, lotes, usuarios, cotizaciones

5️⃣ CAPA DE INFRAESTRUCTURA

Hosting: Vercel | BD: Supabase | Versionado: GitHub | Dominio: parques.trakwerk.com.mx

🔐 Seguridad

  • SessionStorage para autenticación
  • JWT tokens de Supabase
  • Row Level Security en PostgreSQL
  • HTTPS/SSL en todas conexiones
  • Middleware de protección de rutas
  • Variables de entorno para credenciales

📈 Escalabilidad

  • Vercel auto-scaling: 10,000+ usuarios concurrentes
  • Supabase scaling automático
  • Serverless functions sin servidores
  • Capacidad: 1M+ requests/día
