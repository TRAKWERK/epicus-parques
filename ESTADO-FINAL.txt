🚀 INSTRUCCIONES PARA DEPLOYMENT
═════════════════════════════════════════════════════════════════════════════

PASO 1: Sincroniza con GitHub
  cd C:\Users\Usuario\trakwerk-parques
  git push origin main

PASO 2: Crea Supabase (5 min)
  - Supabase.com → Sign Up → Continue with GitHub
  - Crea proyecto "trakwerk-parques"
  - Copia Project URL y Anon Key

PASO 3: Configura Vercel (3 min)
  - Vercel.com → Sign Up → Continue with GitHub
  - New Project → Selecciona trakwerk-parques
  - Agrega variables de entorno (SUPABASE_URL y ANON_KEY)
  - Deploy

PASO 4: (Opcional) Dominio personalizado
  - Vercel → Project Settings → Domains
  - Agrega trakwerk.com.mx/parques
  - Configura DNS (10-15 min)

TOTAL: ~15-20 minutos para estar LIVE en internet
